# Capability Distillation Driven Development (CDDD / C3D)

This folder contains the original whitepaper defining
Capability Distillation Driven Development (CDDD / C3D), a framework
for developing AI-enabled capabilities under uncertainty through
progressive distillation—where not all capabilities are expected to survive.
